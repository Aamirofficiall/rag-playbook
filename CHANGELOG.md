## [0.1.0] - Unreleased

### Added

- Core infrastructure: LLM client, embedder with caching, vector store, chunker, evaluator
- 8 RAG patterns: naive, hybrid search, reranking, parent-child, query decomposition, HyDE, self-correcting, agentic
- CLI commands: compare, run, recommend, ingest, patterns
- Rich terminal output with comparison tables
- Pattern recommendation engine
- Cost tracking per query
- CI/CD pipeline with GitHub Actions
- 226+ unit tests with 80%+ coverage target
