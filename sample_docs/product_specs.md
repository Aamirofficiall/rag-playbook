# Product Specifications

## TechCorp Pro Laptop (Model TC-2026)

### Hardware Specifications

| Component | Specification |
|-----------|--------------|
| Processor | Intel Core i9-14900K, 24 cores, 5.8GHz boost |
| Memory | 64GB DDR5-5600 (2x32GB, expandable to 128GB) |
| Storage | 2TB NVMe PCIe 5.0 SSD |
| Display | 16" Mini-LED, 3456x2234, 120Hz, P3 wide color |
| Graphics | NVIDIA RTX 5090 Mobile, 16GB GDDR7 |
| Battery | 100Wh lithium-polymer, up to 18 hours |
| Weight | 2.14 kg (4.72 lbs) |
| Ports | 3x Thunderbolt 5, 1x HDMI 2.1, 1x SD card reader, 1x 3.5mm audio |

### Software

The TC-2026 ships with TechCorp OS 5.0, which includes:
- Built-in AI assistant powered by on-device LLM
- Automatic cloud backup with 1TB included storage
- TechCorp Suite (word processor, spreadsheet, presentation)
- Security suite with real-time threat protection

### Connectivity

- WiFi 7 (802.11be) with tri-band support
- Bluetooth 5.4
- Optional 5G cellular module (add $199)
- NFC for quick device pairing

### Warranty

All TC-2026 units come with a standard 2-year warranty covering manufacturing defects. Extended warranty options:
- 3-year coverage: $149
- 5-year coverage with accidental damage: $349
- Enterprise fleet pricing available for orders of 50+ units
